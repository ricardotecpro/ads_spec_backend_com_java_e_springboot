# Curso de Desenvolvimento Desktop Java 🖥️

Projeto base para a disciplina de **Desenvolvimento de Sistemas Desktop**. Este repositório contém todo o material didático, aulas, exercícios e slides gerados via MkDocs.

## 🚀 Estrutura do Repositório
*   `/docs/aulas`: Conteúdo teórico das 16 aulas.
*   `/docs/exercicios`: Desafios práticos por aula.
*   `/docs/quizzes`: Verificação de conhecimento.
*   `/docs/slides`: Apresentações Reveal.js.
*   `/docs/projetos`: Mini-projetos evolutivos.

## 🛠 Como rodar localmente
1. Instale o Python 3.x.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Execute o servidor: `mkdocs serve`.
4. Acesse: `http://localhost:8000`.

## 🌐 Deploy
O site é publicado automaticamente no GitHub Pages via branch `gh-pages`.
