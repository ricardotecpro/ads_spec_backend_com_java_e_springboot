# Quiz - Aula 16 ❓

1. **O objetivo do Projeto Integrador é:**
   - (A) Aplicar todos os conhecimentos de POO, MVC, GUI e DB de forma conjunta.
   - (B) Apenas desenhar telas bonitas.
   - (C) Escrever o máximo de código possível em um único dia.
   - (D) Traduzir o sistema para 10 idiomas.
   *Resposta: (A)* - Consolidação do aprendizado prático.

2. **Um requisito técnico essencial do projeto é:**
   - (A) Usar apenas o bloco de notas.
   - (B) Seguir a arquitetura em camadas (MVC e DAO).
   - (C) Não usar classes.
   - (D) Rodar apenas no Linux.
   *Resposta: (B)* - Garante a organização e profissionalismo do software.

3. **A persistência de dados deve ser feita via:**
   - (A) Arquivos .txt simples.
   - (B) Banco de dados relacional com JDBC.
   - (C) Papel e caneta.
   - (D) Memória RAM volátil apenas.
   *Resposta: (B)* - Persistência profissional e escalável.

4. **Na camada View do projeto final, esperamos encontrar:**
   - (A) Comandos INSERT.
   - (B) Componentes Swing (JFrame, JTable, etc).
   - (C) A lógica de cálculo de impostos.
   - (D) O driver JDBC.
   *Resposta: (B)* - Tudo o que diz respeito à interação visual.

5. **A documentação README.md deve conter:**
   - (A) O código-fonte inteiro copiado.
   - (B) Instruções de uso, tecnologias e descrição do sistema.
   - (C) O número do CPF do programador.
   - (D) Apenas imagens.
   *Resposta: (B)* - Guia essencial para quem for rodar o projeto.

6. **Um critério de avaliação importante é a "Boas Práticas", que inclui:**
   - (A) Nomes de variáveis como x1, x2, x3.
   - (B) Código limpo, identado e com nomenclatura clara.
   - (C) Não usar comentários.
   - (D) Colocar todo o código no método Main.
   *Resposta: (B)* - Facilita a leitura e manutenção.

7. **Ao finalizar o sistema, o que deve ser verificado primeiro?**
   - (A) Se a cor do ícone é azul.
   - (B) Se o CRUD está funcionando corretamente (Salvar, Listar, Editar, Excluir).
   - (C) Se o teclado é novo.
   - (D) Se a bateria do notebook está em 100%.
   *Resposta: (B)* - Funcionalidade básica é o coração do sistema.

8. **O Controller no projeto final deve:**
   - (A) Estar vazio.
   - (B) Coordenar as chamadas entre a View e o DAO.
   - (C) Desenhar os botões.
   - (D) Ser um arquivo .css.
   *Resposta: (B)* - Atua como o maestro do fluxo de dados.

9. **O uso do padrão DAO no projeto final permite:**
   - (A) Deixar o código mais lento.
   - (B) Trocar o banco de dados sem mexer na interface gráfica.
   - (C) Que o usuário não precise de teclado.
   - (D) Que o programa rode sem Java.
   *Resposta: (B)* - Flexibilidade arquitetural.

10. **Parabéns! Após concluir este projeto, você terá desenvolvido um:**
    - (A) Site simples.
    - (B) Sistema Desktop profissional completo.
    - (C) Jogo de celular de última geração.
    - (D) Driver de impressora.
    *Resposta: (B)* - Uma base sólida para atuar como desenvolvedor.
