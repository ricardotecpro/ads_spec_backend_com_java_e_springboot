# Quiz - Aula 03 ❓

1. **O conceito de esconder dados internos de uma classe chama-se:**
   - (A) Herança.
   - (B) Polimorfismo.
   - (C) Encapsulamento.
   - (D) Compilação.
   *Resposta: (C)* - O encapsulamento protege o estado interno do objeto.

2. **Qual modificador de acesso restringe o uso apenas à própria classe?**
   - (A) public
   - (B) protected
   - (C) private
   - (D) static
   *Resposta: (C)* - `private` é o nível mais restrito de acesso.

3. **Getters são métodos utilizados para:**
   - (A) Deletar dados.
   - (B) Ler/Obter o valor de um atributo privado.
   - (C) Modificar um valor.
   - (D) Criar a classe.
   *Resposta: (B)* - 'Get' significa pegar/obter.

4. **Setters são métodos utilizados para:**
   - (A) Configurar/Modificar o valor de um atributo privado de forma segura.
   - (B) Apenas imprimir o valor.
   - (C) Somar dois números.
   - (D) Fechar o programa.
   *Resposta: (A)* - 'Set' significa configurar/atribuir.

5. **O construtor de uma classe é executado:**
   - (A) Quando o programa fecha.
   - (B) A cada 10 minutos.
   - (C) No momento da instanciação (`new`).
   - (D) Apenas se houver um erro.
   *Resposta: (C)* - Serve para inicializar o objeto.

6. **Qual o nome correto para o construtor da classe `Cliente`?**
   - (A) public void criar()
   - (B) public Cliente()
   - (C) private static init()
   - (D) public construct()
   *Resposta: (B)* - O construtor deve ter o mesmo nome exato da classe.

7. **A palavra-chave `this` refere-se a:**
   - (A) Outra classe do sistema.
   - (B) Ao objeto atual (a própria instância).
   - (C) Ao banco de dados.
   - (D) Ao sistema operacional.
   *Resposta: (B)* - Usada para diferenciar atributos de parâmetros com o mesmo nome.

8. **Uma vantagem do encapsulamento é:**
   - (A) Deixar o código mais lento.
   - (B) Facilitar a manutenção e validação de dados.
   - (C) Aumentar o tamanho do arquivo .exe.
   - (D) Não usar memória RAM.
   *Resposta: (B)* - Permite validar dados no `set` antes de alterar o atributo.

9. **Se uma classe não define nenhum construtor explicitamente:**
   - (A) O código não compila.
   - (B) O Java fornece um construtor padrão vazio.
   - (C) O objeto nunca pode ser criado.
   - (D) O computador reinicia.
   *Resposta: (B)* - É o chamado "default constructor".

10. **A prática de fornecer construtores diferentes na mesma classe chama-se:**
    - (A) Sobrecarga de construtores.
    - (B) Sobre-escrita.
    - (C) Repetição desnecessária.
    - (D) Herança múltipla.
    *Resposta: (A)* - Overloading permite criar objetos de formas diferentes.
