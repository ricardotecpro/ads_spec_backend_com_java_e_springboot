# Quiz - Aula 11 ❓

1. **JTabbedPane é usado para criar:**
   - (A) Janelas flutuantes.
   - (B) Abas para organizar múltiplos painéis em um mesmo espaço.
   - (C) Tabelas gigantes.
   - (D) Barras de progresso.
   *Resposta: (B)* - Melhora a organização visual de formulários densos.

2. **A barra de menus no topo de uma janela Swing é o:**
   - (A) JMenuBar.
   - (B) JPopupMenu.
   - (C) JToolBar.
   - (D) JHeader.
   *Resposta: (A)* - Contém os JMenus e JMenuItems.

3. **Para exibir uma coleção de dados em colunas e linhas, usamos:**
   - (A) JList.
   - (B) JTable.
   - (C) JGrid.
   - (D) JSpreadsheet.
   *Resposta: (B)* - O componente mais comum para relatórios e listas.

4. **O DefaultTableModel serve para:**
   - (A) Mudar a cor da tabela.
   - (B) Gerenciar os dados (linhas e colunas) da JTable.
   - (C) Criar o banco de dados.
   - (D) Fechar o programa.
   *Resposta: (B)* - A JTable é apenas a face visual; o Model guarda os dados.

5. **Para limpar as linhas de uma tabela, agimos no:**
   - (A) JFrame.
   - (B) TableModel.
   - (C) Monitor do computador.
   - (D) Teclado.
   *Resposta: (B)* - Alterando o modelo, a tabela se atualiza automaticamente.

6. **Validação de formulário serve para:**
   - (A) Deixar o código mais bonito.
   - (B) Impedir que dados incorretos ou incompletos cheguem ao sistema.
   - (C) Mudar o idioma do software.
   - (D) Nada, o banco resolve tudo.
   *Resposta: (B)* - Protege a integridade da aplicação.

7. **Exceptions em Java são usadas para:**
   - (A) Criar abas.
   - (B) Tratar erros em tempo de execução de forma controlada.
   - (C) Deletar arquivos importantes.
   - (D) Melhorar o gráfico dos botões.
   *Resposta: (B)* - Evita que o programa "trave" de forma inesperada.

8. **O bloco `finally` em um try-catch serve para:**
   - (A) Ignorar o erro.
   - (B) Executar um código sempre, havendo erro ou não (ex: fechar arquivos).
   - (C) Reiniciar o computador.
   - (D) Apagar o banco de dados.
   *Resposta: (B)* - Garante a limpeza de recursos.

9. **UX avançada em Desktop inclui:**
   - (A) Usar apenas preto e branco.
   - (B) Teclas de atalho e navegação via teclado (Tab).
   - (C) Remover o mouse do usuário.
   - (D) Não ter menus.
   *Resposta: (B)* - Usuários avançados prezam pela velocidade do teclado.

10. **Podemos customizar a largura das colunas de uma JTable?**
    - (A) Sim.
    - (B) Não, a largura é fixa para sempre.
    - (C) Apenas se a tabela for pequena.
    - (D) Sim, mas apenas no Windows.
    *Resposta: (A)* - Através de métodos no TableColumnModel.
