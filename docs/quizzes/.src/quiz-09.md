# Quiz - Aula 09 ❓

1. **Qual componente é usado para legendas ou rótulos?**
   - (A) JButton.
   - (B) JTextField.
   - (C) JLabel.
   - (D) JInput.
   *Resposta: (C)* - O JLabel exibe texto ou imagem não editável.

2. **Como o usuário insere dados de texto em uma única linha?**
   - (A) Usando JLabel.
   - (B) Usando JTextField.
   - (C) Usando JButton.
   - (D) Usando um Scanner no terminal.
   *Resposta: (B)* - JTextField é o campo de entrada padrão.

3. **Para senhas, o componente ideal é:**
   - (A) JTextField.
   - (B) JSecretField.
   - (C) JPasswordField.
   - (D) JEncrypted.
   *Resposta: (C)* - Ele mascara os caracteres para segurança visual.

4. **Para capturar o clique de um botão, implementamos a interface:**
   - (A) MouseListener.
   - (B) ActionListener.
   - (C) ClickListener.
   - (D) KeyboardListener.
   *Resposta: (B)* - Reage a ações genéricas (clique ou Enter).

5. **JOptionPane serve para:**
   - (A) Criar a janela principal.
   - (B) Exibir mensagens rápidas (alertas, confirmações).
   - (C) Salvar no banco.
   - (D) Desenhar gráficos.
   *Resposta: (B)* - Facilita a criação de diálogos simples.

6. **O método `txtNome.setText("João")` faz o quê?**
   - (A) Apaga o nome João.
   - (B) Define o conteúdo do campo como "João".
   - (C) Salva João no banco.
   - (D) Imprime João no terminal.
   *Resposta: (B)* - Altera o texto exibido no componente.

7. **Qual método verifica se um botão foi pressionado?**
   - (A) run().
   - (B) actionPerformed().
   - (C) onClick().
   - (D) start().
   *Resposta: (B)* - É o método padrão da interface ActionListener.

8. **Para mostrar um erro com ícone vermelho, usamos qual método do JOptionPane?**
   - (A) showMessageDialog(..., ERROR_MESSAGE).
   - (B) showError().
   - (C) printError().
   - (D) showWarning().
   *Resposta: (A)* - O tipo da mensagem define o ícone exibido.

9. **Validação de campos deve ocorrer:**
   - (A) Apenas no banco de dados.
   - (B) No Controller, antes de processar os dados.
   - (C) Apenas na View.
   - (D) Nunca, o usuário sempre digita certo.
   *Resposta: (B)* - Garante que o sistema não tente processar dados inválidos.

10. **UX (User Experience) em formulários desktop envolve:**
    - (A) Usar fontes bem pequenas.
    - (B) Dar feedback claro (mensagens de sucesso/erro).
    - (C) Não usar botões.
    - (D) Esconder os campos do usuário.
    *Resposta: (B)* - Feedback é essencial para uma boa experiência.
