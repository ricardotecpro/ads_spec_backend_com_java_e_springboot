# Quiz - Aula 07 ❓

1. **No padrão MVC, qual componente é responsável por exibir os dados ao usuário?**
   - (A) Model.
   - (B) View.
   - (C) Controller.
   - (D) Banco.
   *Resposta: (B)* - View é a camada de apresentação.

2. **Qual componente decide o que fazer quando um botão é clicado?**
   - (A) Model.
   - (B) View.
   - (C) Controller.
   - (D) BIOS.
   *Resposta: (C)* - O Controller processa os eventos e solicita ações.

3. **Onde ficam validade as Regras de Negócio (ex: cálculo de desconto)?**
   - (A) View.
   - (B) Na área de trabalho.
   - (C) Model ou Controller/Service.
   - (D) No mouse.
   *Resposta: (C)* - Lógica vive longe do visual.

4. **Se o banco de dados mudar de MySQL para Oracle, qual camada mais sofre impacto?**
   - (A) View.
   - (B) Controller.
   - (C) Model/DAO (Persistência).
   - (D) Nenhuma.
   *Resposta: (C)* - É a camada que fala diretamente com o banco.

5. **Qual componente do MVC é considerado o "Pai" ou "Cérebro" do fluxo?**
   - (A) Model.
   - (B) View.
   - (C) Controller.
   - (D) Hardware.
   *Resposta: (C)* - Coordena a interação entre Model e View.

6. **Uma boa prática no MVC é que a View:**
   - (A) Seja a mais complexa possível.
   - (B) Seja "burra", apenas passando dados para o Controller.
   - (C) Não use Java.
   - (D) Seja impressa em papel.
   *Resposta: (B)* - Facilita a reutilização e testes.

7. **Qual diagrama Mermaid melhor representa o fluxo de mensagens entre componentes?**
   - (A) classDiagram
   - (B) sequenceDiagram
   - (C) pie
   - (D) mindmap
   *Resposta: (B)* - Mostra a troca de mensagens ao longo do tempo.

8. **O Model conhece os detalhes da View (cor do botão, etc)?**
   - (A) Sim.
   - (B) Não, o Model é independente da interface.
   - (C) Apenas se o Controller permitir.
   - (D) Sim, para poder se desenhar na tela.
   *Resposta: (B)* - O Model guarda apenas dados e regras lógicas.

9. **O padrão MVC é usado apenas em aplicações Desktop?**
   - (A) Sim.
   - (B) Não, é amplamente usado na Web (Spring MVC, Django, etc).
   - (C) Apenas em aplicativos mobile Android.
   - (D) É um padrão exclusivo da Microsoft.
   *Resposta: (B)* - É um dos padrões mais universais do software.

10. **Refatoração para MVC visa:**
    - (A) Deixar o código mais rápido.
    - (B) Tornar o código mais organizado, testável e manutenível.
    - (C) Gastar mais horas de trabalho.
    - (D) Diminuir o consumo de energia.
    *Resposta: (B)* - Melhora a saúde e longevidade do projeto.
