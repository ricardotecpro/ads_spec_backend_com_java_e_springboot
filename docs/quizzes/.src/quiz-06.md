# Quiz - Aula 06 ❓

1. **Pacotes em Java servem para:**
   - (A) Compactar o código (ZIP).
   - (B) Organizar classes relacionadas e evitar conflitos de nomes.
   - (C) Enviar o código por e-mail automaticamente.
   - (D) Mudar a cor do código.
   *Resposta: (B)* - Funcionam como diretórios organizados.

2. **A convenção de nomes de pacotes segue a ordem:**
   - (A) modulo.projeto.empresa.com
   - (B) com.empresa.projeto.modulo
   - (C) Projeto_Modulo_Empresa
   - (D) 1_2_3_projeto
   *Resposta: (B)* - Domínio invertido é o padrão mundial Java.

3. **Na arquitetura em camadas, a UI (User Interface) deve:**
   - (A) Processar o banco de dados diretamente.
   - (B) Conter toda a lógica de negócio.
   - (C) Apenas coletar dados do usuário e repassar para a próxima camada.
   - (D) Não existir.
   *Resposta: (C)* - Ela deve ser "burra" (sem lógica complexa).

4. **Qual a vantagem de separar Model, View e Controller?**
   - (A) Deixar o projeto com mais arquivos.
   - (B) Facilitar a manutenção e permitir trocas de tecnologias específicas.
   - (C) Ocupar mais espaço no computador.
   - (D) Nenhuma.
   *Resposta: (B)* - Se quiser mudar do desktop para web, o Model permanece igual.

5. **Um projeto bem organizado geralmente tem a pasta principal chamada:**
   - (A) bin
   - (B) src
   - (C) lib
   - (D) doc
   *Resposta: (B)* - 'src' vem de 'source' (fonte).

6. **A camada que cuida das regras de negócio é a:**
   - (A) View.
   - (B) Controller/Service.
   - (C) Banco de Dados.
   - (D) CSS.
   *Resposta: (B)* - O Controller gerencia o fluxo e as regras.

7. **A camada que representa os dados puros (Entidades) é a:**
   - (A) View.
   - (B) Controller.
   - (C) Model.
   - (D) Driver.
   *Resposta: (C)* - Classes como `Produto`, `Cliente`, `Venda`.

8. **O comando Java para usar uma classe de outro pacote é:**
   - (A) include
   - (B) using
   - (C) import
   - (D) require
   *Resposta: (C)* - Indica ao compilador onde a classe está.

9. **O que acontece se mudarmos o banco de dados em um sistema bem arquitetado?**
   - (A) Temos que reescrever a tela inteira.
   - (B) Temos que reescrever apenas a camada de persistência.
   - (C) O sistema para de funcionar para sempre.
   - (D) O teclado para de funcionar.
   *Resposta: (B)* - O impacto é minimizado às camadas inferiores.

10. **A organização de arquivos impacta no trabalho em equipe?**
    - (A) Sim, facilita a divisão de tarefas sem conflitos de arquivos.
    - (B) Não, cada um faz o seu.
    - (C) Apenas se usarmos Git.
    - (D) Apenas se o computador for potente.
    *Resposta: (A)* - Pastas claras ajudam a saber quem mexe em quê.
