# Quiz - Aula 12 ❓

1. **O que é um Banco de Dados Relacional?**
   - (A) Uma lista de contatos no celular.
   - (B) Um sistema que organiza dados em tabelas relacionadas entre si.
   - (C) Um arquivo de texto gigante.
   - (D) Um hardware que guarda CDs.
   *Resposta: (B)* - Utiliza a lógica de conjuntos e relações matemáticas (tabelas).

2. **A sigla SQL significa:**
   - (A) Simple Query Language.
   - (B) Structured Query Language.
   - (C) Software Quality Law.
   - (D) System Quick Login.
   *Resposta: (B)* - Linguagem de Consulta Estruturada.

3. **Qual comando cria a estrutura de uma tabela?**
   - (A) MAKE TABLE.
   - (B) CREATE TABLE.
   - (C) NEW TABLE.
   - (D) INSERT TABLE.
   *Resposta: (B)* - Comando DDL (Data Definition Language).

4. **Para adicionar um novo registro usamos:**
   - (A) ADD.
   - (B) UPDATE.
   - (C) INSERT.
   - (D) SAVE.
   *Resposta: (C)* - Comando DML (Data Manipulation Language).

5. **A chave primária (Primary Key) serve para:**
   - (A) Abrir a porta do servidor.
   - (B) Identificar de forma única cada registro em uma tabela.
   - (C) Mudar o nome do banco.
   - (D) Criptografar os dados.
   *Resposta: (B)* - Garante que não existam registros duplicados idênticos.

6. **O comando `SELECT * FROM produto` faz o quê?**
   - (A) Deleta todos os produtos.
   - (B) Seleciona todas as colunas de todos os registros da tabela produto.
   - (C) Seleciona apenas o primeiro produto.
   - (D) Muda o preço dos produtos.
   *Resposta: (B)* - O asterisco (*) representa "todas as colunas".

7. **SQLite é um banco de dados que:**
   - (A) Precisa de um servidor gigante para rodar.
   - (B) É armazenado em um único arquivo local, ideal para desktop.
   - (C) Só roda em celulares.
   - (D) Não aceita comandos SQL.
   *Resposta: (B)* - Muito prático para pequenos e médios sistemas desktop.

8. **Para atualizar um dado existente, usamos:**
   - (A) CHANGE.
   - (B) MODIFY.
   - (C) UPDATE.
   - (D) REFRESH.
   *Resposta: (C)* - Permite alterar valores de colunas específicas.

9. **O comando `DELETE FROM cliente WHERE id = 1` irá:**
   - (A) Deletar todos os clientes.
   - (B) Deletar apenas o cliente que tem o ID igual a 1.
   - (C) Deletar apenas a coluna ID.
   - (D) Erro de sintaxe.
   *Resposta: (B)* - A cláusula WHERE é fundamental para filtrar a operação.

10. **Modelagem de dados é a fase onde:**
    - (A) Programamos as telas.
    - (B) Definimos as tabelas, colunas e relacionamentos do banco.
    - (C) Escolhemos a cor do ícone.
    - (D) Instalamos o Windows.
    *Resposta: (B)* - É o projeto da base de dados.
